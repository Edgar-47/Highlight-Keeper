Que hace

Funciones pensadas para lectura activa y trabajo real.

La extension esta pensada para que la pagina siga siendo la protagonista, pero con una capa adicional de memoria, accion y contexto visual.

01

Subrayado persistente

Resalta texto seleccionado y recuperalo al volver a la misma pagina sin rehacer el trabajo.

02

Notas post-it flotantes

Coloca recordatorios o comentarios en cualquier punto de la pantalla y sigue editandolos cuando quieras.

03

Colores utiles

Usa paletas predefinidas o colores personalizados para diferenciar ideas, tareas y prioridades.

04

Persistencia por contexto

El contenido se guarda asociado a la URL y puede evolucionar hacia dominio, cuenta o sincronizacion.

Como la hemos hecho

Arquitectura limpia para crecer sin deuda innecesaria.

El proyecto se ha separado en modulos claros para que cada parte haga solo su trabajo y podamos ampliar funciones sin romper el flujo principal.

Popup

Centro de acciones

Desde la popup eliges color, creas notas, ejecutas resaltados y controlas la experiencia principal.

Content Script

Interfaz sobre la pagina

Pinta highlights, renderiza post-it, escucha interacciones y mantiene la capa visual sincronizada.

Background

Coordinacion y menus

Gestiona menus contextuales, mensajes y acciones globales propias de la extension.

Storage

Persistencia local

Guarda highlights, notas, colores, posiciones y preferencias usando la API de almacenamiento.

Stack tecnico

Lenguajes y tecnologias utilizados

La base es deliberadamente sencilla y mantenible: tecnologia web estandar, buena separacion y cero dependencias innecesarias para la pagina de presentacion.

HTML

Estructura de popup, landing page y elementos base de la experiencia visual.

CSS

Sistema visual, estados, responsive, animaciones y estetica moderna para web y extension.

TypeScript

Codigo fuente tipado para records, mensajes y logica de highlights y notas.

JavaScript

Runtime de la extension y tambien esta landing, compatible con GitHub Pages sin backend.

Chrome Extension API

Manifest V3, popup, background, content scripts, context menus y storage local.

GitHub

Repositorio, documentacion y punto natural para desplegar esta pagina de forma publica.

Flujo de trabajo

Leer, marcar, anotar y volver mas tarde sin perder nada.

  1. Seleccionas texto o creas una nota desde la popup o el menu contextual.
  2. La extension guarda contenido, color, posicion, tamano y metadatos utiles.
  3. Al regresar a la pagina, el sistema intenta restaurar highlights y post-it automaticamente.
  4. Puedes editar, recolorear, mover o eliminar contenido cuando quieras.
