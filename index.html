Extension privada para navegador

Una herramienta para leer, subrayar y dejar notas sin perder el contexto.

Esta extension de Chrome permite guardar subrayados persistentes por pagina y crear notas post-it flotantes sobre cualquier web. El objetivo es convertir una lectura normal en un espacio personal de trabajo, analisis y recordatorios.

Ver codigo Perfil de Edgar-47

Incluye

  • Subrayado persistente por URL
  • Colores predefinidos y personalizados
  • Notas adhesivas flotantes
  • Arrastre, redimensionado y titulo editable
  • Guardado local con chrome.storage

Para que sirve

Una capa personal sobre cualquier pagina web.

La extension esta pensada para personas que investigan, estudian, comparan informacion o simplemente quieren volver a una pagina y encontrar sus ideas justo donde las dejaron. En vez de depender solo de marcadores o capturas, el contenido relevante se mantiene visible sobre la propia web.

Funciones

Lo mas importante

  • Subrayar texto seleccionado y recuperarlo al volver a la pagina.
  • Cambiar colores y evitar duplicados innecesarios.
  • Crear notas post-it movibles y editables.
  • Guardar texto, color, posicion, tamano e identificador unico.
  • Trabajar en local sin publicar la extension.

Como la hemos hecho

Arquitectura modular para crecer bien.

Se ha construido sobre Manifest V3, separando responsabilidades entre popup, content script, background service worker, almacenamiento y modulos independientes para subrayado y notas. La idea es mantener el proyecto facil de ampliar sin mezclar demasiada logica en un solo archivo.

Stack tecnico

Lenguajes y tecnologias utilizadas

HTML

Estructura de popup, pagina informativa y base visual de la extension.

CSS

Interfaz visual, sistema de color, layout responsive y apariencia de post-it digital.

TypeScript

Codigo fuente mantenible con tipos claros para highlights, mensajes y notas.

JavaScript

Version ejecutable local usada directamente por la extension cargada en Chrome.

Chrome Extension API

Manifest V3, content scripts, popup, context menus, service worker y storage local.

chrome.storage.local

Persistencia de subrayados, colores, notas, posiciones, tamano y ajustes del usuario.

Capas

Que hace cada parte

  • Popup: crear resaltados, seleccionar color, crear notas y lanzar acciones.
  • Content script: pintar highlights y renderizar notas sobre la pagina.
  • Background: menus contextuales y coordinacion con pestañas.
  • Storage: persistencia por URL y ajustes de usuario.

Estado del proyecto

Hecho para evolucionar

Aunque funciona como herramienta privada/local, el proyecto ya tiene una base preparada para seguir creciendo: exportacion de datos, autenticacion, sincronizacion, panel lateral o incluso una futura version comercial si se decide publicar.
